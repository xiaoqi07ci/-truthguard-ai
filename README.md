## 🎯 What it does
**Real-time AI hallucination detector!** Analyzes text and flags:
- ✅ **Verified** (Green) - Factual 
- ⚠️ **Suspicious** (Yellow) - Unrealistic claims
- ❌ **Hallucinated** (Red) - Impossible facts
